# 🌎 Escola de Idiomas & Cursos Livres - Landing Page

## 📖 Sobre o Projeto
Uma landing page dinâmica, moderna e focada em conversão, desenvolvida para escolas de idiomas, professores particulares e centros educacionais. O grande objetivo deste projeto é transmitir autoridade na metodologia de ensino, apresentar os cursos de forma clara e facilitar a captação de novos alunos (leads) através do agendamento de aulas experimentais.

Este projeto compõe meu portfólio profissional de desenvolvimento Front-End e demonstra minha habilidade em criar interfaces voltadas para o nicho da educação, onde a confiança e a clareza nas informações são decisivas para a matrícula.

## ✨ Principais Funcionalidades
* **Design Acolhedor e Educacional:** Layout estruturado para transmitir profissionalismo e incentivar o aprendizado, utilizando uma tipografia clara e um visual limpo.
* **Mobile-First & Responsividade:** Como muitos jovens e pais pesquisam cursos pelo celular, a página foi otimizada para oferecer uma navegação perfeita e rápida em telas menores.
* **Vitrine de Cursos:** Seção dedicada para detalhar os idiomas ensinados (Inglês, Espanhol, etc.), as modalidades (presencial ou online) e as faixas etárias.
* **Geração de Leads (CTA de Matrícula):** Botões de "Agende uma Aula Experimental" ou "Fale com um Consultor" estrategicamente posicionados, conectando o futuro aluno diretamente ao WhatsApp da escola.
* **Alta Performance:** Código leve para garantir um carregamento imediato, evitando que o usuário perca o interesse antes mesmo de conhecer a escola.

---

## 🛠️ Tecnologias Utilizadas
Projeto construído "na unha", utilizando as bases sólidas da web para garantir máxima performance e total controle sobre a estrutura visual, sem o peso de frameworks externos:

* **HTML5:** Estrutura semântica focada em acessibilidade e em um excelente ranqueamento no Google (SEO local), vital para atrair alunos da região.
* **CSS3:** Estilização responsiva, uso de Grid/Flexbox para organizar os blocos de informações e animações que tornam a experiência mais fluida.
* **JavaScript (Vanilla):** Dinamismo na interface, ideal para implementar sanfonas de perguntas frequentes (FAQ) ou interações na grade de cursos.
